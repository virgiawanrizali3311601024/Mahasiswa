Technical details
------------------

### Run

- clone repo from git
- run `npm install`
- run `npm start`
- open `http://localhost:8080` in a browser 

### Deploy

- run `npm run build`
- copy "index.html", "data" and "codebase" folders to the server

### Other commands

- `npm run lint` - will validate all js code in the project
